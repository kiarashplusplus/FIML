# REST API Reference

FIML provides a comprehensive REST API for financial data access.

## Base URL

```
http://localhost:8000
```

For production, use HTTPS with your domain.

## Interactive Documentation

- **Swagger UI**: [http://localhost:8000/docs](http://localhost:8000/docs)
- **ReDoc**: [http://localhost:8000/redoc](http://localhost:8000/redoc)

## Common Endpoints

For complete API documentation with interactive examples, visit the Swagger UI at `/docs`.
